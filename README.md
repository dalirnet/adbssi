# adbssi

[ADB](#) [s](#)hell [s](#)creencap [i](#)nterface

![carbon](carbon.png)

### Install

```bash
npm install -g adbssi
```

### Usage

```bash
adbssi screenshot.png
```

#### Requirement

-   [adb](https://adbdriver.com/downloads/) (connected to device)
-   [node](https://nodejs.org/en/download/) (version >=14)
